# Tiny+

A simple compiler for language Tiny+

### Language Definition

see `Tiny+EBNF.md`

### Usage

```
Usage: ./tiny <file> [options]
Options:
  -d            Print syntax tree
  -h            Print help
  -o <file>     Place the output into <file>
```
